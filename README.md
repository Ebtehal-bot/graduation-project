# 2. نبذة عن المشروع

نظام **كفيل لرعاية وكفالة الأيتام** هو منصة إلكترونية تهدف إلى تحويل عمليات إدارة كفالة الأيتام من الإجراءات التقليدية إلى نظام رقمي متكامل.

يوفر النظام أدوات لإدارة بيانات الأيتام والكفلاء والكفالات والمدفوعات والتقارير والإشعارات والصلاحيات، بالإضافة إلى تطبيق جوال يساعد الكفلاء على متابعة الكفالات الخاصة بهم.

يتكون المشروع من تطبيقين رئيسيين:

1. لوحة تحكم ويب لإدارة النظام.
2. تطبيق جوال للكفلاء.



# 3. المشكلة التي يعالجها المشروع

تعاني بعض المؤسسات الخيرية من صعوبة إدارة بيانات الأيتام والكفلاء باستخدام الطرق التقليدية، مما يؤدي إلى:

- صعوبة متابعة بيانات الأيتام والكفلاء.
- تأخر تحديث معلومات الكفالات.
- صعوبة استخراج التقارير والإحصائيات.
- ضعف التواصل بين الجمعية والكفلاء.
- الحاجة إلى نظام مركزي وآمن لإدارة البيانات.



# 4. أهداف المشروع

- إنشاء نظام إلكتروني لإدارة كفالة الأيتام.
- تنظيم بيانات الأيتام والكفلاء في قاعدة بيانات مركزية.
- تسهيل إدارة عمليات الكفالة والمدفوعات.
- توفير تطبيق جوال للكفلاء.
- تحسين عملية التواصل والمتابعة.
- توفير تقارير وإحصائيات تساعد الإدارة في اتخاذ القرار.

| الطبقة | التقنية | الاستخدام |

| Admin Panel | Filament v2 | بناء لوحة الإدارة |

| Mobile | Flutter | تطوير تطبيق الجوال |
| Mobile Language | Dart | لغة تطبيق الجوال |
| Database | MySQL | تخزين البيانات |
| Networking | Dio | الاتصال بالـ API |
| State Management | Riverpod | إدارة حالة التطبيق |
| Tools | VS Code / Android Studio | بيئة التطوير |
| Tools | XAMPP | تشغيل قاعدة البيانات محليًا |
| Version Control | Git / GitHub | إدارة الإصدارات |
