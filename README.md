# نبذة عن المشروع

منصة **كفيل لرعاية وكفالة الأيتام** هي منصة إلكترونية متكاملة تهدف إلى تسهيل وإدارة عمليات كفالة الأيتام بشكل رقمي وآمن.

يتكون المشروع من:

- لوحة تحكم إلكترونية لإدارة النظام.
- تطبيق جوال للكفلاء.
- قاعدة بيانات مركزية.
- واجهات برمجية (REST API) لربط التطبيق بلوحة التحكم.

توفر المنصة بيئة متكاملة لإدارة بيانات الأيتام والكفلاء والكفالات والمدفوعات والإشعارات والتقارير والصلاحيات، بما يسهم في تحسين كفاءة إدارة أعمال المؤسسات الخيرية.

# تشغيل المشروع

## أولاً: تشغيل لوحة التحكم

قم بتشغيل خدمات XAMPP:

- Apache
- MySQL

ثم نفذ الأوامر التالية:

```bash
composer install
```

```bash
php artisan storage:link
```

```bash
php artisan migrate
```

```bash
php artisan serve --host=0.0.0.0 --port=8000
```

لوحة التحكم:

```
http://127.0.0.1:8000/admin
```
